# Scrapling Agent Skill

该 skill 遵循 [AgentSkill](https://agentskills.io/specification) 规范，因此可被 [OpenClaw](https://github.com/openclaw/openclaw)、[Claude Code](https://claude.com/product/claude-code) 及其他 agent 工具读取。它将文档网站的大部分内容封装为 Markdown，agent 无需猜测即可使用。

可用于回答关于 Scrapling 的约 90% 常见问题。我们已在 [OpenClaw](https://github.com/openclaw/openclaw) 和 [Claude Code](https://claude.com/product/claude-code) 上测试，如遇问题请提交 [工单](https://github.com/D4Vinci/Scrapling/issues/new/choose)，或加入我们的 [Discord 服务器](https://discord.gg/EMgGbDceNQ)。

## 安装

可使用此[直接链接](https://github.com/D4Vinci/Scrapling/raw/refs/heads/main/agent-skill/Scrapling-Skill.zip)下载 skill 的 ZIP 包。我们会尽量在此页更新所有可用安装方式。

### Clawhub
若你使用 [OpenClaw](https://github.com/openclaw/openclaw) 或 [Claude Code](https://claude.com/product/claude-code)，可通过 [Clawhub](https://docs.openclaw.ai/tools/clawhub) 直接安装：

```bash
clawhub install scrapling-official
```

或访问 [Clawhub 页面](https://clawhub.ai/D4Vinci/scrapling-official)。
