# Response 类

`Response` 类封装所有 Fetcher 返回的 HTTP 响应，提供对状态码、请求头、正文、Cookie 以及用于解析的 `Selector` 的访问。

可如下导入 `Response` 类：

```python
from scrapling.engines.toolbelt.custom import Response
```
